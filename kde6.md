## 6.9

KDE runtime inherits the 
[Freedesktop 24.08 components](https://gitlab.com/freedesktop-sdk/freedesktop-sdk/-/tree/release/24.08/elements/components?ref_type=heads) 
and provides 
[KDE runtime 6.9 components](https://invent.kde.org/packaging/flatpak-kde-runtime/-/blob/qt6.9/org.kde.Sdk.json.in?ref_type=heads), 
which contain extra modules or an updated version of inherited modules.

**module name**

**kirigami addons**

_kirigami-addons_ (Add-ons for the Kirigami framework) is part of the KDE runtime starting with version 6.9. It is no longer necessary to compile this module, unless the project requires a specific version or configuration of this dependency.

**ocean sound theme** 

_ocean-sound-theme_ (Ocean Sound Theme for Plasma) is part of the KDE runtime starting with version 6.9. It is no longer necessary to compile this module, unless the project requires a specific version or configuration of this dependency.
